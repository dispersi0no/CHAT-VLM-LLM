## 🎯 Как использовать

### Режим OCR (Распознавание текста)

1. Откройте http://localhost:8501
2. Выберите **"📄 Режим OCR"**
3. Выберите режим:
   - **vLLM** (dots.ocr) - уже запущена
   - **Transformers** (Qwen3-VL-2B) - работает
4. Загрузите изображение
5. Нажмите **"Обработать"**

### Режим чата (Вопросы об изображении)

1. Откройте http://localhost:8501
2. Выберите **"💬 Режим чата"**
3. Загрузите изображение
4. Задайте вопрос
5. Получите ответ

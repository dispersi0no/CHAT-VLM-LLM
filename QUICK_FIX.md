# ⚡ Быстрое исправление переключения моделей

## 🎯 Проблема
Кнопка переключения моделей не работает → ошибка "образ не найден"

## ✅ Решение (выберите одно)

### Вариант 1: Загрузить образ (5-10 мин, один раз) 📥

```bash
python pull_vllm_image.py
```

**Результат:** Полная функциональность, можно создавать новые модели

---

### Вариант 2: Использовать существующие контейнеры (мгновенно) ⚡

```bash
python fix_model_switching.py
```

**Результат:** Быстрое переключение между уже созданными моделями

---

## 🚀 После исправления

```bash
# Запустить систему
python start_system.py

# Открыть браузер
http://localhost:8501

# Переключать модели через UI ✅
```

---

## 📋 Проверка

```bash
# Проверить образы
docker images vllm/vllm-openai

# Проверить контейнеры
docker ps -a

# Проверить GPU
nvidia-smi
```

## 💡 Рекомендация

**Для постоянного использования:** Вариант 1 (загрузить образ)  
**Для быстрого теста:** Вариант 2 (существующие контейнеры)
